Parking Management System:
This is a SQL-based Parking Management System project developed using MySQL. The project manages vehicle records, parking slots, parking records, and payment details.

The system includes:

* Vehicle management
* Parking slot management
* Payment tracking
* Parking reports
* Revenue calculation

SQL concepts used in this project:

* CREATE DATABASE
* CREATE TABLE
* INSERT INTO
* SELECT queries
* UPDATE queries
* JOIN queries
* GROUP BY
* Aggregate Functions

Tables included:

1. Vehicles
2. Parking_Slots
3. Parking_Records
4. Payments

Technologies Used:
- SQL Server
